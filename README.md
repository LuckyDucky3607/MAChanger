### [+] Installation
 - `git clone https://github.com/LuckyDucky3607/MAChanger`
 - `cd MAChanger`

### [+] Usage
 - `python MAChanger.py --interface {interface} --mac {new_mac}`
 - `python MAChanger.py --interface {interface} --default {to restore the default MAC}`

### [+] ONLY WORKS WITH LINUX
